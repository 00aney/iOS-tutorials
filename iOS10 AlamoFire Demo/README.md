#iOS10 AlamoFire Demo

###Install cocoapods
```
sudo gem install cocoapods
```

```
pod setup
```


In project root directory,
```
pod init
```

###Edit Podfile
```
open -a Xcode Podfile
```

In Podfile,

added ```pod 'Alamofire', '4.3.0'```

###pod install
```
pod install
```


###Usage
```swift
import Alamofire
```
